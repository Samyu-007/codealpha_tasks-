# CodeAlpha Java Internship Projects

This repository contains four Java projects completed as part of the CodeAlpha Java Programming Internship.

## Project 1: Student Grade Tracker

### Description
A Java console application to manage student grades.

### Features
- Enter student names and marks
- Calculate average marks
- Display highest and lowest marks

---

## Project 2: Stock Trading Platform

### Description
A simple Java console application that simulates stock trading.

### Features
- Display available stocks
- Buy stocks
- Update remaining balance

---

## Project 3: AI ChatBot

### Description
A rule-based chatbot that answers basic user questions.

### Features
- Interactive chat
- Predefined responses
- Exit option

---

## Project 4: Hotel Reservation System

### Description
A Java application to book and cancel hotel rooms.

### Features
- View room availability
- Book rooms
- Cancel bookings
